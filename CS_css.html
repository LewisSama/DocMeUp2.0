Selecteurs

On peut définir un sélecteur CSS comme une ensemble de mots-clés et/ou de symboles, qui permet d'identifier les éléments qui vont profiter du style CSS. Les sélecteurs CSS trient les éléments de la page selon leur type, certains de leurs attributs, selon leur imbrication dans le code de la page ou encore selon l'interaction avec l'utilisateur pour les liens par exemple (liste non exhaustive).

  • Comment ça marche

    Selecteur {proprieté: valeur;}

  • Selecteur universel

    * { font-family: serif; }

  • Sélecteur de type

    h2 { font-size: 150%; }

  • Sélecteurs d'attributs

    Les deux attributs les plus couramment utilisés sont class et id

    class ou ID {proprieté: valeur;}

  • Sélecteur de classe

    .nom_de_classe { /* déclaration(s) */ } élément.nom_de_classe { /* déclaration(s) */ }

  • Sélecteur d'identifiant

    #nom_id { /* déclarations */ }

  • Sélecteur d'attribut

    <input name="saisie"> input[name]{border:1px solid silver}

  • Les sélecteurs hiérarchiques

    on peut spécifier des règles pour les éléments contenus à l'intérieur de certains autres éléments sans ajouter de classe ni d'identifiant

  • Le sélecteur de descendant

    #wikipedia h1 {... }

  • Le sélecteur d'enfant

    permet de désigner un élément par filiation directe à un autre élément

    #wikipedia > h1 {...}

  • Le sélecteur de frère adjacent

    permet de définir un style pour un élément enfant suivant immédiatement un autre élément enfant du même élément parent.

    h1 + h2 {...}

  • Pseudo-classes des liens

    a:link {text-decoration: none;} a:visited {text-decoration: line-through;}

  • Pseudo-classes d'interaction avec l'utilisateur

    a:hover {text-decoration: underline;} input:focus {color: #400; background-color: #fff;} a:active {text-decoration: none;}

Box-Model

Tous les éléments sont considérés comme des boîtes (box). Chacune de ces boîtes est constituée d’un contenu, d’un espacement intérieur, d’une bordure, et d’une marge externe. Voici les propriétés CSS qui permettent de déterminer les dimensions, la couleur, le style de chacun de ses constituants :

  • width et height largeur et hauteur du contenu (texte, image, etc.)

  • padding espacement intérieur, entre le contenu et la bordure

  • border bordure (ou encadrement)

  • margin marge externe, espace (transparent) entourant le tout
