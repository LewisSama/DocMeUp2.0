Selecteurs

On peut définir un sélecteur CSS comme une ensemble de mots-clés et/ou de symboles, qui permet d'identifier les éléments qui vont profiter du style CSS. Les sélecteurs CSS trient les éléments de la page selon leur type, certains de leurs attributs, selon leur imbrication dans le code de la page ou encore selon l'interaction avec l'utilisateur pour les liens par exemple (liste non exhaustive).

  • Comment ça marche

    Selecteur {proprieté: valeur;}

  • Selecteur universel

    * { font-family: serif; }

  • Sélecteur de type

    h2 { font-size: 150%; }

  • Comment ça marche

    Selecteur {proprieté: valeur;}

  • Selecteur universel

    * { font-family: serif; }

  • Sélecteur de type

    h2 { font-size: 150%; }
